# aoc22

My Advent of Code answers for 2022, with a helper script.

## Usage

Run today's code
```sh
npm start
```

Run day 2's code
```sh
npm start -- 2
```

Watch today's code
```sh
npm run dev
```

Watch day 2's code
```sh
npm run dev -- 2
```
